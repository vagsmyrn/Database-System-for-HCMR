Introduction

Compatibility

Browser

It is recommended to use Mozilla Firefox to view Large Pelagic Database. It is also compatible with Google Chrome and Microsoft Internet Explorer . It is also recommended that you have updated these programs to their latest versions.

Insert Expedition Data

1. Vessel Information

  1. AMAS search box is used to search through all inserted vessels. You may search by their AMAS or by their name. Any value left in this input field won't be submitted to the expedition data as this field is used ONLY for search!
  2. AMAS in this field you input the desired AMAS of the vessel you want to assign the expedition data to. If there are more than one vessels in the expedition you add more fields by click on the "+" button!
  3. Number of vessels and "+" button are used to specify the number of vessels that are assigned to the expedition you are going to insert to the database. The number of vessels field is automatically increased depending on how many times you click the "+" button.
    IMPORTANT! If you add more input fields than the vessels you want to insert into the expedition data you have to refresh the page as for security reasons we didn't add a "-" button to remove AMAS fields. So you are advised to use the "+" button carefully and double-check how many vessels you want to add in each expedition data.

2. Expedition Data

Here you input the details of the expedition data. On the Catch section if you want to add more than one species, you click the respective "+" button (NOTE: if you fill the first input fields and then click the "+" button the Number and Weight fields will go to the next fields and the first ones will be empty! So you have to double-check if the selected species and the inputted amounts are valid for each species!)

3.Catch Measurements

First you have to select the species you want to add measument values for. As different species have different input fields! This is important because if you input i.e. values for an Albacore and you haven't selected "Albacore" in "Select Species" dropdown menu, any values inputted in wrong fields will not be submitted and stored in the database!
